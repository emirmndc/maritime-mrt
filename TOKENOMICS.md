## Wallet Role Clarification
- **Deploy Main Account:** Used for deployment and setup tasks. Not intended as the main long-term storage wallet.
- **Owner Hold:** Founder allocation wallet. Intended for long-term alignment with the project.
- **Liquidity Pool Wallet:** Used to add/manage DEX liquidity positions.
- **Treasury:** Reserved for project operations and strategic needs.
- **Community:** Reserved for community growth and ecosystem development.

## Holder Concentration Context (Early-Stage Project)
At the early stage of a project, scanners may show concentrated holdings because supply is still held across project wallets before broader market distribution develops.

This does **not** by itself imply malicious behavior. The project aims to improve transparency by:
- publishing wallet roles,
- documenting tokenomics openly,
- clarifying roadmap vs live functionality,
- and improving market metadata visibility.
